Privacy Policy

Super Google AI Assistant — Chrome Extension

📅 Last updated: March 14, 2026

👋 Introduction

Super Google AI Assistant ("the Extension") is a Chrome browser extension that provides AI-powered assistance for managing Google Workspace services including Google Sheets, Drive, Docs, Gmail, Calendar, YouTube, and Google Apps Script. This Privacy Policy explains how we collect, use, and protect your information.

📋 Information We Collect

Data Stored Locally (on your device)

  • Google OAuth token (for API authentication)
  • API keys you provide (Gemini, Claude) — stored in Chrome's local storage
  • GitHub and Vercel tokens (if you connect them)
  • Selected AI model preference
  • Chat history cache

Data Stored in Your Google Account

  • Chat history — saved to a Google Sheet named "AI Assistant - Chat History" in your own Google Drive
  • Images from chat — uploaded to a folder named "AI Assistant Images" in your own Google Drive
  • Settings/preferences — stored in the Settings tab of the same Google Sheet

Data We Do NOT Collect

  • ❌ We do NOT collect personal information on any external server
  • ❌ We do NOT track your browsing activity
  • ❌ We do NOT send your data to any third-party analytics service
  • ❌ We do NOT store your Google password

✅ Your Data Stays Yours

All data is stored either locally on your device (Chrome storage) or in your own Google Drive. We do not operate any backend server that collects or stores user data. The Extension communicates directly with Google APIs and AI providers using your own credentials.

🔑 Google API Scopes & Permissions

The Extension requests the following Google OAuth scopes to provide its features:

Scope Purpose Type
userinfo.email Identify your Google account Basic
userinfo.profile Display your name and avatar Basic
spreadsheets Read/write Google Sheets data Sensitive
drive.file Access files created by the extension or selected via file picker Sensitive
documents Read/write Google Docs content Sensitive
gmail.readonly Read email messages (no modification) Sensitive
gmail.send Send emails on your behalf Sensitive
gmail.compose Create and manage drafts Sensitive
calendar Read/write Google Calendar events Sensitive
script.projects Manage Google Apps Script projects Sensitive
youtube Manage YouTube account and videos Sensitive
youtube.upload Upload videos to YouTube Sensitive

⚠️ All API calls are made directly from your browser to Google servers. No data passes through any intermediary server.

🌐 Third-Party Services

The Extension may interact with the following third-party services when you explicitly configure them:

  • Google Gemini API — for AI responses (requires your own API key)
  • Anthropic Claude API — for alternative AI responses (requires your own API key)
  • GitHub API — for code repository management (optional, requires your token)
  • Vercel API — for deployment management (optional, requires your token)

These services have their own privacy policies. We encourage you to review them. Your API keys and tokens are stored locally and never shared.

🔒 Data Security

  • All communication with Google APIs uses HTTPS encryption
  • OAuth tokens are managed by Chrome's built-in identity system
  • Sensitive keys (API keys, tokens) stored in Google Sheets are lightly obfuscated (Base64 encoding)
  • No data is transmitted to any server owned or operated by us
  • You can revoke access at any time via Google Account Permissions

🗑️ Data Deletion

You can delete all data associated with this extension at any time:

  • Local data: Uninstall the extension from Chrome — all local storage is automatically deleted
  • Google Drive data: Delete the "AI Assistant - Chat History" sheet and "AI Assistant Images" folder from your Google Drive
  • Revoke access: Remove the extension from your Google Account Permissions

👶 Children's Privacy

This Extension is not intended for use by children under the age of 13. We do not knowingly collect any personal information from children.
