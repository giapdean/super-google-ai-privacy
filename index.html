❓ FAQ

Câu hỏi thường gặp

Có! Tất cả công cụ Google (Sheets, Drive, Docs, Gmail, Calendar, YouTube, Apps Script) đều miễn phí vĩnh viễn. Chỉ GitHub và Vercel yêu cầu gói Pro.
Bạn cần ít nhất 1 API Key: Gemini (miễn phí tại aistudio.google.com) hoặc Claude (trả phí tại console.anthropic.com). Extension không lưu key trên server — tất cả lưu local trên máy bạn.
Hoàn toàn. Extension chạy trực tiếp trong Chrome, gọi API Google/AI thẳng từ trình duyệt. Không có server trung gian nào xử lý dữ liệu của bạn. API Key lưu trong Chrome Storage local.
Google Chrome, Microsoft Edge, và tất cả Chromium-based browsers (Brave, Arc, Opera, Vivaldi...).
AI tự phân bổ token budget dựa trên model: Gemini hỗ trợ ~200KB data/lần, Claude ~60KB. Với tính năng analyze_in_chunks, AI tự chia nhỏ data lớn thành nhiều phần để xử lý.
Chuyển khoản ngân hàng qua mã QR SePay. Hệ thống tự xác nhận thanh toán trong 3-5 giây sau khi chuyển tiền. Hỗ trợ tất cả ngân hàng Việt Nam.
